# `celebrate` 测试文档

## 场景信息

- 场景名：`celebrate`
- 对应脚本：`Motions/08_celebrate/scene_script.py`
- 主要目标：验证 Mira 的庆祝段落是否能形成展位情绪峰值

## 测试目标

本轮主要确认：

1. 起手、上摇、下摇、彩灯、dance、减速收尾是否完整
2. 音乐和动作能否配合
3. 即便音频失效，场景是否仍能成立

## 测试前准备

- 蓝牙音箱在线
- 音量先调到可控范围
- 起始姿态建议为 `neutral`

## 测试步骤

1. 将灯复位到 `neutral`
2. 点击 `跳舞模式 / celebrate`
3. 观察开头预热是否清楚
4. 观察上摇和下摇是否有明显情绪爆发
5. 观察多色灯效是否切换正常
6. 观察音乐是否正常播放
7. 观察结尾减速、摇头、回正是否自然

## 预期结果

- 整体节奏应显著比其他场景更兴奋
- 灯光变化明显，但不应乱闪到失控
- 有音乐时应更像情绪高潮
- 没音乐时也不应整幕垮掉

## 失败信号

- 没有明显情绪峰值
- 灯光、动作、音乐彼此脱节
- 结尾突然结束，没有减速收尾
- 音箱问题导致整幕中断

## 失败回退

- 点击 `Stop -> Neutral`
- 如果音频异常，下一轮先关闭音乐只测动作和灯效
