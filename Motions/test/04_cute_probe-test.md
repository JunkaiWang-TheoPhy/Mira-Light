# `cute_probe` 测试文档

## 场景信息

- 场景名：`cute_probe`
- 对应脚本：`Motions/04_cute_probe/scene_script.py`
- 主要目标：验证 Mira 的卖萌节奏是否成立

## 测试目标

本轮主要确认：

1. 轻点头、左右找角度、探头、缩回、再探出的顺序是否清楚
2. 动作是否显得可爱，而不是像测试舵机
3. 中段探头是否有“胆小试探”的感觉

## 测试前准备

- 起始姿态建议为 `neutral`
- 测试环境尽量安静，方便纯看动作节奏

## 测试步骤

1. 将灯复位到 `neutral`
2. 点击 `卖萌 / cute_probe`
3. 观察开头轻点头是否自然
4. 观察左右找角度是否小而有节奏
5. 观察中间关节上提和下放是否明显
6. 观察探头动作是否先慢慢出去，再快速缩回
7. 观察最后再探一次并稳定收尾

## 预期结果

- 整体节奏要慢，不能太急
- 轻点头和左右找角度都应克制
- 探头要有“先试一下、被吓到、再试一下”的层次
- 收尾应稳定，不留奇怪姿态

## 失败信号

- 节奏太快，像机械摆头
- 探头没有停顿，看不出“胆小”
- 上提和下放不明显
- 结束姿态不稳

## 失败回退

- 点击 `Stop -> Neutral`
