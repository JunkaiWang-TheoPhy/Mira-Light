# `sz_sigh_comfort` 固定动作实现指导

## 当前目标

先做一版“叹气 -> 暖光 -> 轻低头”的固定安慰动作。

## 固定版实现原则

- 不依赖实时麦克风
- 主持人口播先建立语境
- 用暖灯和轻微机械动作完成主要表达

## 优先复用的板端脚本

- `send_uart3_led_cmd.py breathe ...`
- `send_uart3_led_cmd.py all ...`
- `servo_2_nod_1900_2200.py`

## 第一版推荐实现

1. 主持人说“你对着它叹一口气”
2. 灯光切暖色呼吸
3. 轻点头或轻低头一次
4. 保持一小段停顿

## 为什么适合先做固定版

因为这个场景的重点不是复杂动作，而是：

- 情绪方向是否正确
- 表达是否克制

这两点靠固定动作已经能验证。

## 后续升级方向
