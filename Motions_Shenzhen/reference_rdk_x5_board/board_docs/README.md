# Board Docs

这个目录放的是从地瓜板 `/home/sunrise` 拉回来的板端说明文档和协议资料。

## 当前包含

- `UART1_SERVO_README.md`
- `UART3_LED_PROTOCOL_README.md`
- `UART3_LOOPBACK_NOTES.md`
- `灯头指令.md`
- `舵机协议手册(191218-0923)(1).pdf`
- `磁编码sts-内存表解析_220714_v3.xlsx`

## 它们分别有什么用

### `UART1_SERVO_README.md`

最适合回答：

- 板上的总线舵机现在怎么控
- 设备是 `/dev/ttyS1` 还是别的
- 波特率是多少
- 现有命令行怎么读位置、发移动、改 ID

### `UART3_LED_PROTOCOL_README.md`

最适合回答：

- 灯头控制现在走哪条串口
- 命令行脚本怎么发
- 线怎么接

### `UART3_LOOPBACK_NOTES.md`

更偏联调记录，用于解释：

- UART3 回环测试时要注意什么

### `灯头指令.md`

这是 LED 灯头文本协议文档。

对于深圳版很重要，因为很多“情绪光”设计最后都要落到这里的命令能力上。

### `舵机协议手册...pdf` 与 `磁编码...xlsx`

这是底层舵机协议真值。

如果你后面要超出当前 Python 脚本封装层，自己做更底层控制，这两份资料会变得非常重要。

## 对深圳版的意义

这个目录不是给主控台看的，而是给真正落地真机控制看的。

如果未来你要：

- 重写 `send_uart1_servo_cmd.py`
- 校准深圳版动作
- 把 LED 情绪表达做实

这些文档会是必须回看的真值层。
