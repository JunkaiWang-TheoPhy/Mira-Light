# `standup_reminder` 测试文档

## 场景信息

- 场景名：`standup_reminder`
- 对应脚本：`Motions/06_standup_reminder/scene_script.py`
- 主要目标：验证 Mira 是否能像宠物一样做“蹭蹭提醒”

## 测试目标

本轮主要确认：

1. 三次蹭蹭的节奏是否清楚
2. 点头和摇头是否都能看懂
3. 整体是不是“可爱提醒”，不是机械撞击

## 测试前准备

- 起始姿态建议为 `neutral`
- 主持人口播建议先交代“假装你已经坐很久了”

## 测试步骤

1. 将灯复位到 `neutral`
2. 主持人口播进入提醒语境
3. 点击 `久坐检测：蹭蹭 / standup_reminder`
4. 观察是否先转向用户
5. 观察三次蹭蹭是否节奏分明
6. 观察后段双点头是否清楚
7. 观察末段轻摇头和回位是否自然

## 预期结果

- 蹭蹭应是三次有节奏的前顶，不应像乱抖
- 双点头应能一眼看懂是“是的”
- 轻摇头应克制，像“好吧”
- 收尾应回到自然位

## 失败信号

- 三次蹭蹭之间没有明显节奏差
- 点头不明显
- 摇头太夸张，破坏角色感
- 没有主持人口播时，场景显得莫名其妙

## 失败回退

- 点击 `Stop -> Neutral`
