# `touch_affection` 测试文档

## 场景信息

- 场景名：`touch_affection`
- 对应脚本：`Motions/03_touch_affection/scene_script.py`
- 主要目标：验证 Mira 是否像“主动蹭手”，而不是机械前探

## 测试目标

本轮主要确认：

1. 前探、下沉、轻蹭、追手、回位这几段是否完整
2. 动作是否亲近、克制，不像攻击或撞击
3. 如果有真实输入，左右侧响应是否合理

## 测试前准备

- 起始姿态建议为 `neutral`
- 真机测试时，确保测试者手部不要离灯头过近
- 如果当前没有真实 `hand_near` 输入，可先用导演台手动触发

## 测试步骤

1. 将灯复位到 `neutral`
2. 点击 `摸一摸 / touch_affection`
3. 观察是否先轻柔前探
4. 观察身体是否有轻微下沉，形成“钻到手下”的感觉
5. 观察中段轻蹭是否小而连续
6. 观察末段是否有一次追手动作
7. 观察是否回到自然等待位

## 预期结果

- 前探应该柔和，不应像扑过去
- 轻蹭应是小幅上下/左右组合
- 追手应该只有一下，不应追个不停
- 收尾应回到自然工作姿态

## 失败信号

- 前探过猛，看起来像撞击
- 蹭动幅度过大，失去亲密感
- 没有追手或追手过长
- 无人触发时也会“空蹭”

## 失败回退

- 优先 `Stop -> Neutral`
- 如果真机位置有风险，立即人工拉开手并停止 scene
